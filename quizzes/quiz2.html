Python OOP Quiz: Test Your Knowledge

Q1. What does the __init__ method do in Python?

Answer: b) Initializes a class instance.

The __init__ method in Python is a constructor used to initialize an instance of a class when it is created.

Q2. Which of the following is NOT a feature of Object-Oriented Programming in Python?

Answer: d) Compilation

Python is an interpreted language, and compilation is not a feature of its OOP design.

Q3. What is the purpose of the super() function in Python?

Answer: a) To access the parent class constructor or methods.

The super() function is used to call a parent class's methods or constructor in Python.

Q4. Which of the following is used to define a block of code in Python?

Answer: c) Indentation

Python uses indentation to define blocks of code, which enhances readability and structure.

Q5. What is the output of print(type([])) in Python?

Answer: a) <class 'list'>

The function type([]) returns the class type of the empty list, which is list.

Q6. What is the correct way to declare a dictionary in Python?

Answer: a) my_dict = {"key": "value"}

Dictionaries in Python are defined using curly braces { } with key-value pairs.

Q7. Which of the following is not a valid Python data type?

Answer: c) Array

While Python supports arrays through libraries like numpy, it does not have a built-in Array type; the primary data structures are lists, tuples, and dictionaries.

Q8. What will len("Hello, World!") return?

Answer: c) 13

The len() function returns the number of characters in the string, including spaces and punctuation.

Q9. What does the is keyword in Python do?

Answer: b) Checks if two objects are the same instance.

The is keyword checks if two objects refer to the same memory location, not just if their values are equal.

Q10. Which of the following methods can be used to convert a string to lowercase in Python?

Answer: b) lower()

The lower() method is used in Python to convert a string to all lowercase letters.

Q11. What is the purpose of the self keyword in Python classes?

Answer: c) It is a reference to the instance of the class.

The self keyword is used in Python to refer to the instance on which a method is being called, allowing access to its attributes and other methods.

Q12. What is a decorator in Python?

Answer: a) A function that modifies another function or method.

A decorator is a Python function that takes another function as an argument and extends or modifies its behavior without explicitly modifying its code.

Q13. What is the difference between a shallow copy and a deep copy?

Answer: b) Shallow copy duplicates references, while deep copy duplicates objects.

A shallow copy creates a new object but inserts references into it for the original objects, while a deep copy creates a new object and recursively copies all objects within.

Q14. How do you create a virtual environment in Python?

Answer: b) Use the command python -m venv [env_name].

To create a virtual environment in Python, use the command python -m venv [env_name]. This sets up an isolated environment for Python projects.

Q15. What is the purpose of the with statement in Python?

Answer: c) To ensure proper acquisition and release of resources.

The with statement in Python is used for resource management, ensuring that resources like file handles are properly released after use.
